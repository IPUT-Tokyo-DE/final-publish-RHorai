# Sword Game

## ゲームの内容
剣を持って森にいる魔物を退治しよう

![画面イメージ](docs/images/game_image01.png)

## 操作説明
左右移動
　左 A ←
　右 D →
ジャンプ W ↑
急落下 S ↓
攻撃 スペースキー

通常攻撃
地面にいるときに攻撃 正面に攻撃範囲がある
空中攻撃
空中にいるときに攻撃 上方向に攻撃範囲がある
